◆ startCal()

void MagSensor::startCal ( uint16_t  maxSample = 500)

Start the calibration process for the mag sensor.

This starts the magnetometer calibration process. The sensor must be moved in a figure 8 pattern for the duration of the calibration. The calibration will take maxSample samples.

Parameters
maxSampleThe maximum number of samples to take.

bool_t MagSensor::stopCal ( bool_t  cancel,
Vector3 *  biasCorrection,
Matrix3x3 *  transformCorrection 
)

Stop the calibration process for the mag sensor.

This stops the magnetometer calibration process. If cancel is false the aquired data will be used to calculate the calibration values.

Parameters
cancelIf true the calibration process will be cancelled and the calibration values will not be updated.
[out]biasCorrectionVector3* This can be nullptr. As calibrations are compounded this bias correction can be used to correct the aquired data.
[out]transformCorrectionMatrix3x3* This can be nullptr. As calibrations are compounded this transform correction can be used to correct the aquired data.
Returns
bool_t True if the calibration was successful.
